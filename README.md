# An image organizer JavaFx app

### Compiling
- Source code is under `src`.
- Main is under `src\view\Main.java`


### Opening the Program
- Locate jar file: `cd classes/artifacts/ProjectPhase2_jar`
- Execute jar file: `java -jar  ProjectPhase2.jar`
